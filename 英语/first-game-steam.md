# 游戏策划

1. 首先要有个主基地，以此为据点，种田丰收，类似星露谷，可以通过雇佣或者其他交换的方式来让别人来协助你打理后勤系统

2. 邀请他人来加入你的军团，可以有多种邀请的方式
   - 友谊，梦想，崇拜
   - 赏金
   - 强制

3. 军团必须要保证后勤，否则会饿死，后勤提供也分质量
   - 好的质量，战斗力高
   - 低质量知识保证饿不死
   - 士气也能提高战斗力
     - 领导魅力
     - 友谊值

4. npc也需要种田养活自己
   - 要有一套经济体系，npc如果饿死了会复活吗，还是真的死了，人口怎么解决

5. 在大地图上有多个从其他地方来的入口，就是用来解决人口问题的，可以根据城市宜居度选择是否定居，定居了就是本领地一员
   - 可以安排自己人在人流密集地方介绍自己领地的优势，让他们搬迁过去，形成发展的人口红利

6. 打怪物也会有仇恨值，如果你有某种怪物一直不打他们，而你的竞争对手一直打他们，则你们在一起的时候，怪物优先攻击仇恨打的人

7. 世界中的角色，可以分成普通居民，铁匠，士兵，武术家，猎人，药剂师，服装师，领主，除了居民，每一种角色都有技能，

   如果居民跟着铁匠工作就能增加铸造的技能经验，另外一个角色可以学习有多个技能。

8. 如果你成功的让一个角色加入你，他就处于待命状态，你就可以对他发出指令，如果他处于饥饿状态下，就会乱跑，会自己去寻找食物，也会降低对你的好感度

9. 有战争迷雾，盗贼可以隐身，隐身有距离，靠近别人就会有幻影，越近越明显，宝藏猎人
